Durante este curso aprenderemos los fundamentos de HTML, y cómo utilizarlos de forma correcta

Hemos visto las siguientes etiquetas:

<html> y </html>
Lo utlizamos para definir el documento HTML.
Hay que incluirlo siempre en nuestros documentos HTML.
<html> y </html>
Dentro de este elemento se encuentra el código HTML que el navegador mostrará al usuario.
Se pueden definir estilos en los atributos de este elemnto.
<p> y </p>
Con este par de etiquetas definimos un párrafo.
Nos sirve para estructurar el texto.
No olvidar cerrar el elemento con la etiqueta correspondiente
<hr/>
Esta etiqueta simple nos permite insertar una línea horizontal.
Nos sirve para separar cabecera, cuerpo y pie del documento.
No olvidar cerrarla.
